About

PatchIt! is a Python 3 application written by Triangle717  and is the standard and simple way to package and install LEGO® Racers mods.

The Plan

Although LEGO® Racers modding is still in it's early stages, many mods are already being developed, and there needs to be a standard way to install them. PatchIt!, which was heavily influenced and based on Patchman!, a mod installer for the 1999 Data Design Interactive PC game LEGO® Rock Raiders, fills that void.
