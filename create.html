Creating a Patch

To create a PatchIt! Patch, load PatchIt! and press the c key, and enter your Patch's Name, Version, Author, and Description. Point it to your Patch files, and PatchIt! will compress them into an .PiA archive and write the details to a .PiP file.  The Patch files need to be laid out a certain way for PatchIt! to properly work. The excellent tutorial Setting Up/Installing a LEGO® Racers Mod will explain everything.

PatchIt! 1.1.1 Stable and later automatically filters your files and stops any "illegal" files from being compressed into the Patch. Complete details on this process can be found on GitHub.

Both files use your Patch's name and version for the filename, and will be located in the same folder as your uncompressed Patch files. Check out the PiP Format documentation for complete details.
