Introduction

This is the Readme for PatchIt!, standard and simple way to package and install mods for the 1999 High Voltage Software PC game LEGO Racers, written purely in Python 3.

Usage

Patch Installation

To install a PatchIt! compatible mod, simply load PatchIt!, press the 'i' key, select the PiP file, and after reading the mod information, confirm the installation. PatchIt! will automatically install it into your LEGO Racers game. If you need a patch to install, jump down to the Resources section to download one! If you are unsure about everything you need to do to successfully install a Patch, an awesome tutorial by JimbobJeffers on what to do is in the Resources section.

Patch Creation

To create a PatchIt! compatible mod, load PatchIt! and press the 'c' key. Type your mod's name, version, author, and description. Point it to your modded files, and PatchIt! will compress them into a normal ZIP archive and write the details to a PiP file. As of PatchIt! 1.0.1 Stable, if Thumbs.db is found in your modded files, it will automatically deleted to avoid an error when someone installs your patch. Don't worry, Windows will automatically recreate it. ;)
Both items use your mod's name and version for the filename, and will be located in the same folder as your uncompressed modded files. Check the the PiP Format documentation (linked in the Resources section below) for complete details.

If you have made a LEGO LOCO Patch, you will need to enter the screen resolution you used to create the map. This is important, for if it is played in a different resolution, some elements may be cut off, destroying the hard work you put into making it.

First Run

Upon running PatchIt! for the first time, you will be asked to set up the required Settings. The steps below will walk you through the process, or you can figure it out yourself.
You will only need to do this once, as PatchIt! is smart enough to know if you have already run it.

Changing your LEGO Racers Installation

If you need to change your defined LEGO Racers installation, you can do it in a minute. Load PatchIt! , press the 's' key, then the 'r key, and point it to your LEGO Racers installation. If it confirms a valid installation, press the 'n' key, and you're off! If it can't be verified, you'll need to enter it again.

Good To Know!

A valid LEGO Racers installation is defined by the existence of three files: LEGORacers.exe, LEGO.JAM, and GolDP.dll

Changing your LEGO LOCO Installation

If you need to change your defined LEGO LOCO installation, you can do it just as easily as you would change your LEGO Racers installation. Just load PatchIt!, press the 's' key, then the 'l' key and point it to your LEGO LOCO installation. If it confirms a valid installation, press the 'n' key, and you're off! If a valid installation cannot be can't verified, you'll need to enter it again.

Good To Know!

A valid LEGO LOCO installation is defined by the existence of three files: Exe\Loco.exe, Exe\LEGO.INI, and art-res

Support

To request a feature, submit a code patch, or download a new release of PatchIt!, please go to https://github.com/le717/PatchIt.
To submit a bug report, please go to https://github.com/le717/PatchIt/issues, state the bug, and attach PatchIt.log, located at %ProgramFiles%\PatchIt\Logs.
Attaching this file will ensure the bug will be found a lot easier, and a bug fix will be released much faster.
