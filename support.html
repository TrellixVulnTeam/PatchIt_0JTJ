Support

To request a feature, submit a code fix, or download a new release of PatchIt!, please go to http://github.com/le717/PatchIt.

To submit a bug report, please go to http://github.com/le717/PatchIt/issues, describe the bug and the steps to recreate it, and attach your PatchIt.log file, located at %ProgramFiles%\PatchIt\Logs.

Attaching the log will ensure any bugs will be found quicker and easier, and a bug fix will be released sooner.

Installing PatchIt!

There are two ways to install PatchIt! onto your computer.

1. You can use the installer, and install PatchIt! to your computer the standard way. This will install PatchIt! to your Program Files location with a start menu and  Desktop shortcut (although both are optional).

2. PatchIt! is also portable, meaning that it can be used on a USB flash drive. To setup a portable installation of PatchIt!, go to the folder above the directory of the PatchIt! installer. Hold Shift and Right-Click on the folder. Press “Open command window here”. Type the name of the installer file, and append /portable=1.
