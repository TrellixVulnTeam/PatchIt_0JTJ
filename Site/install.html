Installing a Patch

To install a PatchIt! Patch, simply load PatchIt!, press the 'i' key, select the PiP patch, and after reading the Patch information, confirm the installation.

PatchIt! will automatically install it into your LEGO® Racers or LEGO® LOCO installation, depending on the game the Patch was created for. If you need a Patch to install, or you are unsure about everything you need to do to successfully install one, go to the Resources page.
