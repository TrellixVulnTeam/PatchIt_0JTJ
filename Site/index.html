PatchIt! The standard and simple way to package and install LEGO® Racers mods.

Welcome to the PatchIt! readme. Here, you will find information, support, and help about PatchIt! PatchIt! was created by Triangle717 and is continually being updated! PatchIt! has the ability to package and install modifications for the 1999 video game LEGO® Racers. It also has support for the 1998 video game LEGO® LOCO.
