Creating a Patch

To create a PatchIt! compatible mod, load PatchIt! and press the 'c' key. Type your mod's name, version, author, and description. Point it to your modded files, and PatchIt! will compress them into a PatchIt! Archive (PiA) file and write the details to a PatchIt! Patch (PiP) file. PatchIt! 1.1.1 Stable and later automatically filter your files and stops any "illegal" files from being compressed into the Patch. Complete details on this process can be found on GitHub.

Both items use your mod's name and version for the filename, and will be located in the same folder as your uncompressed modded files. Check the the PiP Format documentation for complete details.

 

If you have made a LEGO® LOCO Patch, you will need to enter the screen resolution you used to create the map. This is important, for if it is played in a different resolution, some elements may be cut off, destroying the hard work you put into making it.
