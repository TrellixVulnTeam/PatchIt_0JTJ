Creating a Patch

To create a PatchIt! Patch, load PatchIt! and press the 'c' key. Select the video game it is creaed for, and enter your Patch's Name, Version, Author, and Description. Point it to your Patch files, and PatchIt! will compress them into a PatchIt! Archive (PiA) file and write the details to a PatchIt! Patch (PiP) file.

PatchIt! 1.1.1 Stable and later automatically filters your files and stops any "illegal" files from being compressed into the Patch. Complete details on this process can be found on GitHub.

Both items use your Patch's name and version for the filename, and will be located in the same folder as your uncompressed Patch files. Check the the PiP Format documentation for complete details.

 

If you have made a LEGO® LOCO Patch, you will need to enter the screen resolution you used to create the map. This is important, for if it is played in a different resolution, some elements may be cut off, destroying the hard work you put into making it.
