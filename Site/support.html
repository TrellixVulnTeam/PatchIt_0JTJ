Support

To request a feature, submit a code patch, or download a new release of PatchIt!, please go to http://github.com/le717/PatchIt.

To submit a bug report, please go to http://github.com/le717/PatchIt/issues, state the bug, and attach your PatchIt.log file (located at C:\Program Files\PatchIt\Logs).

Attaching this file will ensure the bug will be found a lot easier, and a bug fix will be released much faster.
